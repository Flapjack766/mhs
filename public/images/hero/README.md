# Hero Images Directory

This directory contains all images used in the hero slider on the homepage.

## Current Images:
- **hero-timber-bg.jpg** - Main hero background image
- **wood-beams.jpg** - Structural wood beams
- **wood-planks.jpg** - Quality wood planks  
- **construction-site.jpg** - Construction projects
- **timber-bg.jpg** - Timber background

## Image Specifications:
- **Format**: JPG
- **Recommended Size**: 1920x1080 pixels (16:9 ratio)
- **Quality**: High resolution for web display
- **File Size**: 500KB - 2MB per image

## Adding New Images:
1. Add your image to this directory
2. Update the `heroImages` array in `app/[locale]/page.tsx`
3. Follow the naming convention: descriptive-name.jpg

## Usage:
These images are automatically displayed in a rotating slider on the homepage hero section, changing every 4 seconds with smooth transitions.
